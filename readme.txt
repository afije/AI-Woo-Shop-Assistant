Enhance your WooCommerce store with AI-powered product recognition and suggestions.

== Description ==

AI Shop Assistant for WooCommerce is a powerful plugin that uses artificial intelligence to analyze images and suggest relevant products to your customers. With this tool, you can provide a more interactive and personalized shopping experience.

Features:
* Real-time image analysis
* Automatic product suggestions based on image content
* Customizable shop assistant button
* Easy integration with your existing WooCommerce store

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-shop-assistant` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> AI Shop Assistant screen to configure the plugin.
4. Enter your OpenAI API key in the settings.

== Frequently Asked Questions ==

= Does this plugin require an OpenAI API key? =

Yes, you need to obtain an API key from OpenAI to use the image analysis features of this plugin.

= Is this plugin compatible with all themes? =

The plugin should work with most WordPress themes, but some customization may be required for optimal display depending on your theme's structure.
